Reference Types (Objects)

Reference Types "Objects"
are a non-primitive value and when assigned to a Variable, the variable is given a reference to the value.
Object ;iterals, arrays and function are all reference types

Static Typing vs Dynamic Types

JavaScript is a dynamically-typed Language :- this means, we do not explicitly define the types for our variable, Many other languages are staticaly-typed such as C, C++, JavaScript

TypeScript is a superset od JavaScript, which allows staticaly-typing. this can make your code more verbose and less prone to erros. TypeScript may be something you want to look into later on.
